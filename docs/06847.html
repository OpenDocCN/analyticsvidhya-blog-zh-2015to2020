Python编程语言中有四种集合数据类型:列表、元组、集合和字典。在本节中，我们将学习list。

目录

列表是有序且可改变的集合。

在Python中，创建列表的方法是将所有元素放在方括号[]中，用逗号分隔。它可以有任意数量的元素，并且它们可以是不同的类型(整型、浮点型、字符串型等。).

示例:

my_list = []                            #empty list
