# 什么是可教机器？

Teachable Machine 是一个基于网络的工具，它使创建机器学习模型变得快速、简单，并且每个人都可以访问。你训练计算机识别你的图像、声音和姿势，而无需编写任何机器学习代码。然后，在你自己的项目、网站、应用程序和…中使用你的模型
