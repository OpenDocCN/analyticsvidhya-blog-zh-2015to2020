**什么是帕金森病？**

帕金森病是一种与年龄相关的中枢神经系统神经退行性疾病。这是仅次于阿尔茨海默病的第二常见疾病，影响大脑中产生多巴胺的神经元。它影响运动，引起震颤和僵硬。据估计，全世界有 7000 万到 1000 万人患有帕金森病。
