## 什么是机器学习？

机器学习是给计算机编程的科学(和艺术),以便它们可以从数据中学习。

**一般定义:**

机器学习是在没有明确编程的情况下赋予计算机学习能力的研究领域。[阿瑟·塞缪尔，1959 年]
