如何理解docker-compose的up vs run vs exec命令之间的区别

问题

您开始使用容器，并注意到Docker Compose提供了多种运行容器的方式，即up、run和exec。两者的区别是什么？你什么时候想用其中一个而不是另一个？

解决办法

大多数情况下，您很可能希望启动您的docker-compose.yml中列出的所有服务，并让容器运行它们的默认命令，因此您可能希望使用up。
