标准普尔 500 是一个股票市场指数，衡量在美国证券交易所上市的 500 家大公司的股票表现。一家公司的市场资本总额的计算方法是将当前股票价格乘以发行在外的股票。

***用于分析的数据——苹果、亚马逊、微软&谷歌***

数据集中的特征概述
