网飞如何利用数据科学成为 1000 亿美元的公司

网飞/有趣的事实

🧐网飞公司从数据中获取洞察力，以提升其用户体验！

网飞有超过 1 . 83 亿用户，而且这些数字还在持续成倍增长。

尽管网飞在过去几年中采取了一些大胆而有争议的措施，如决定在 2016 年全面封锁VPN，以及同年备受反对的提价——因此失去了大量用户——但它仍然是最大的订阅流媒体平台。

怎么会？

所有的荣誉都归于数据科学:大数据和算法。

正如托马斯·莱德曼所说，“哪里有数据的烟雾，哪里就有商业的火焰。”

简介:

网飞是当今最大的订阅流媒体服务提供商。

它使用数据科学不仅获得新用户而且增强现有用户的用户体验。

网飞知道用户想看什么，甚至比用户自己更早。多亏了数据科学。

它使用各种传统的商业智能工具，以及大数据技术来提出改进的算法，以增强用户体验，从而提高其利润。

显示像纸牌屋和橙色是新的黑色不是赌博，而是精心合成数据的结果。

个性化的观察列表和艺术品是秘密成分。

一些令人难以置信的事实

由于它的算法，网飞每年节省大约 10 亿美元！

早在 2006 年，一家 DVD 邮寄公司推出了 Netflix 奖 。它向任何人或任何团体提供 100 万美元，只要他们能创造出最佳算法，根据之前的评级来确定订户对任何节目或电影的喜欢程度。

网飞上 80%的内容都是基于推荐系统的。

仅在 2019 年，它就赚了 200 亿美元。学分？数据科学。

其 49%的用户都是付费订阅的；也就是说，他们不共享密码和帐户。大多数用户为他们的订阅付费。

在看完所有这些事实之后，你可能会奇怪，一个普通的 DVD 邮寄公司怎么会成长为如此巨大的成功——同类中最大和最重要的。继续读下去，你很快就会发现。

Gary King(哈佛大学教授)说，“大数据不是关于数据的。这是关于分析的。”

网飞如何从成堆的数据中获得卓越的用户体验和金钱

虽然数据是一种基本需求，但没有经过适当处理的数据只是一堆无用的信息。

当您处理这些数据以获得有用的见解，从而带来客户满意度和利润时，真正的游戏就开始了。

网飞使用各种传统的商业智能工具(如 Teradata 和 MicroStrategy )并结合现代大数据技术，如 Hadoop 、 Hive 等。因此，它形成了预先决定用户最有可能观看什么的算法。同样，它也使用了很多开源软件。

Justin Ward ( 数据科学与工程团队经理，网飞)说:“大数据对一个数据科学团队来说或多或少都意味着什么。它比你习惯处理的存储空间更大，所以你需要寻找替代方案。这比你以前必须做的处理要多得多。你需要更多的技巧来找到相关性和惊喜。许多可以处理这种分析的工具并不像你期望的那样工作。这是一个新世界，你需要用适合自己的方式去探索它。”

因此，基本上，比拥有数据更重要的是知道如何引导数据，为您的企业和客户带来有利的结果。

个性化是关键！

网飞有近 2 亿用户，他们每天每时每刻都在产生海量数据。

不利用这些疯狂的数据为自己谋利是愚蠢的。

不同于那些对观众的需求和喜好只有模糊概念的电视网络，网飞拥有一个强大的用户个人数据库。这都要感谢数据科学。

说网飞有一亿八千三百多万个版本不会错。

这种个性化不仅增强了用户体验，也有助于预测哪些连续剧或电影将为他们赢得最多的订户，从而获利。

个性化视频排名:

对于完全相同的类型，不同的用户有完全不同的视频推荐。这是因为网飞为每位订户定制了整个网飞系列。因为他们知道，每个人都有独特的品味。

甚至趋势部分的内容也是个性化的。

网飞的艺术品个性化

网飞的个性化推荐系统的主要目的是在正确的时间把最合适的书放在正确的观众面前。

但是推荐的作用并没有到此为止。标题应该足够吸引人，让浏览者点击它，看它能提供什么。这就是艺术品发挥作用的地方。

艺术作品向观众提供了一种关于标题的视觉证据，从而帮助他们决定是否观看该节目或电影。

网飞不仅仅向 1.83 亿人提供产品。它以 1.83 亿种不同的方式提供产品。T9】

你看到的艺术品之所以在那里，是因为它包含了你可能会觉得有吸引力的东西。它可能会突出显示你可能认识的演员的脸，也可能会显示你可能会感兴趣的场景。

例如，让我们考虑死亡诗社——一个有几部浪漫电影观看历史的人可能会看到突出伊桑·霍克的艺术作品。同时，一个搜索艺术史或历史电影的人可能会展示洞穴中的一个场景。并且，一个有高中戏剧观看历史的人，可能从教室里看到一个场景。

现在的问题是，他们是如何实现的？通过 AVA ，简称艺术品视觉分析。

AVA(艺术品视觉分析):

这是一种算法，可以浏览可用的视频，并选择最适合它们的图像。

AVA 考虑了几件事，比如场景照明、场景中突出显示的内容、物体放置、背景等。

“如果星巴克的秘密是当你拿到拿铁咖啡时的一个微笑…我们的秘密是网站适应了个人的口味。”
— 雷德·哈斯汀斯(网飞 CEO)

如何个性化？

向如此广泛的受众提供个性化的内容并不容易，尤其是当受众每天都呈指数级增长时。

如上所述，有各种软件用于创建算法，为所有不同的用户个性化网飞，但这里有一些要点，其技术在个性化时要注意:

你看什么样的内容。

你在哪天和什么时间看。

在什么设备上观看节目或电影。

内容是暂停、倒带还是快进？

搜索、滚动和浏览行为

你的位置等

预绿灯。贴绿灯。以及个性化的作用。

到现在为止，你必须确定这样一个事实，网飞所做的每一件事都是基于算法和数据科学。

网飞的红极一时的系列如纸牌屋、橙色是新的黑色、皇冠、等。，不是赌博，而是综合了数据见解的结果。

甚至在播出之前，网飞就知道这些节目会非常受欢迎。怎么会？你很快就会发现。

对《纸牌屋》更深入的分析

自 2013 年开播以来，《纸牌屋》一直保持着 8.8 IMBD 评分。不仅如此，网飞还在推出《纸牌屋》的三个月内，仅在美国就获得了200 万订户，其余的在国际上也获得了100 万订户。

但与当时流行和安全的趋势不同，网飞没有推出试点。相反，它甚至在第一集播出之前就提前委托制作了两季节目！

听起来像是大赌博？商业风险？

不是。网飞有数据。它知道它的3300 万用户喜欢导演大卫·费舍尔的作品《社交网络》。它还知道，以凯文·史派西为原型的电影在其平台上表现良好。锦上添花的是，这个节目的英国版在网飞很受欢迎。因此，他们对这场演出的成功充满信心。

直到现在，数据科学还没有被用于个性化。很快，个性化部分就会发挥作用。

个性化预告片:现在，网飞已经投资制作即将热播的节目，并对其制作发出了绿色信号，现在它将制作个性化预告片。

是的，网飞为《纸牌屋》的预告片做了十个不同的剪辑，考虑到了用户的个性化偏好。

所有这些结合在一起，提供了最佳的用户体验，并确保网飞保持其领先地位。

网飞:万金油？

网飞不仅使用数据科学制作、流式传输和推荐个性化的节目和电影，还发布和流式传输尽可能多样化的内容。

除了许多受欢迎的网飞原著，如《巫师》、《奇怪的事》等。它购买了《宋飞正传》等热门老节目的流媒体播放权。

它还播放纪录片，如获得奥斯卡提名的《美国工厂》。

网飞还与超级明星制作人签署了独家大单，如珊达·瑞姆斯、T21、瑞恩·墨菲等。

这种内容的巨大多样性不仅仅是为了灵活变通，或者因为它可以购买或生产所有这些内容。这是因为它服务于几乎 2 亿的观众，知道谁喜欢什么。再次感谢数据科学。

每个有手机的人都有一个屏幕和互联网接入。这是我们的目标市场。世界的品味，世界的时间，是我们所追求的。
—雷德·哈斯汀斯(网飞 CEO)

但是他们是如何做到这一点的呢？

还是那句话，数据科学！

网飞非常明智地使用以下(仅举几例)和许多其他方法来确保卓越的用户体验及其持续增长:

Durid :

Apache Druid 是一个高性能的实时分析数据库。它专为快速查询和接收事务的工作流而设计。Druid 擅长即时数据可见性、即席查询、运营分析和处理高并发性。”—德鲁伊.伊奥

Durid 有助于为网飞提供实现高质量用户体验所需的各种实时数据和分析。

Durid 为任何数量的工作负载提供了出色的性能。

Python:

Python 的推特账号标记了网飞，并在推特上说 Python 帮助网飞在不到 10 分钟内重新路由了近三分之一的互联网流量。

网飞使用 Python 做很多事情，包括以下内容:

  1. 维护信息安全。
  2. 分析流量分布和运行数据。
  3. 原型可视化工具等。

元卡特:

网飞对大量数据进行操作。这些数据涌入几个不同的平台，如亚马逊 S3 、杜里德、红移等。为了保持互操作性，也就是说，为了在这些不同的平台之间保持信息的顺畅交换和使用，它需要 Metacat。

Metacat 为所有数据存储提供集中的元数据访问。

结论

数据科学不仅在网飞的基本运作中发挥着重要作用，还能帮助其维护和发展用户。

网飞是利用数据科学实现卓越业务的典范。

从一家名不见经传的 DVD 租赁公司起步，到成为最大的流媒体订阅服务公司，网飞走过了漫长的道路。这种成功并非完全是运气使然。对数据进行适当的综合才是神奇之处。数据科学万岁！

但这并不总是一帆风顺的。通常，技术会带来特定的问题。但是网飞成功地解决了所有的问题，并且表现得相当不错。

虽然你可能没有网飞那么多的资源或数据，但你也可以利用数据科学来给你的业务提供更好的方向和增长。

数据科学不仅是客户满意的途径，也是提供商发展的途径。
