## 介绍

冠状病毒是一种 RNA(核糖核酸)病毒，由大约 27-32kb 的正义单链 RNA 组成。冠状病毒属于冠状病毒科，包括α、β、δ和γ冠状病毒。已知该病毒会感染广泛的宿主，包括人类、其他哺乳动物和鸟类。在印度，13.4 亿人已经按照印度政府发布的指导方针，遵循一级防范、保持社交距离和其他预防措施。因此，我试图涵盖新冠肺炎对印度人口的影响。
