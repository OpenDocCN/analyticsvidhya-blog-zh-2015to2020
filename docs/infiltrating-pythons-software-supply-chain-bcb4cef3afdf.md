ZDNet 两天前发布了[这篇](https://www.zdnet.com/article/two-malicious-python-libraries-removed-from-pypi/)有趣的帖子，标题为“*两个恶意的 Python 库被抓到窃取 SSH 和 GPG 密钥*”，为 2020 年及以后的事情做准备。
