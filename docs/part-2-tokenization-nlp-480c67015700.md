# 什么是标记化(NLP)？

**标记化**是将字符串、文本标记化或拆分成标记列表的过程。我们可以把记号看作是部分，就像单词是句子中的记号，句子是段落中的记号一样。

# 为什么是标记化(NLP)？

正如我们所知，拆分句子或单词的过程被称为标记化。我们做标记化是因为如果我们将有不同的单词和句子，那么我们可以触及每个单词和句子，并且可以从每个单词或句子中获得洞察力。这就是为什么标记化对我们来说非常重要。
