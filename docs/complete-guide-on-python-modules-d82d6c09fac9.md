在 Python 中，模块只是一个包含 Python 代码的文件。我们使用模块将应用程序分解成更小的块，以提供代码的可重用性。

我们可以在一个模块中定义我们最常用的函数并导入它，而不是将它们的定义复制到不同的程序中。

让我们用一些可重用的函数创建一个模块，文件名为`operations.py`
