# 迭代程序

> 在 Python 中，**迭代器**是实现**迭代器**协议的对象，该协议由 __iter__()和 __next__()方法组成。— [python 文档](https://docs.python.org/3/howto/functional.html#iterators)

*   迭代器是表示数据流的对象。
*   它一次返回一个元素的数据。
