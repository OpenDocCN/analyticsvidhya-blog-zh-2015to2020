# 计算机视觉:从暑期实习生项目到重新定义人工智能未来

大约在 1966 年:人工智能的马文·明斯基&麻省理工学院实验室的名气给了大学生杰拉德·让伊·萨斯曼一个暑期项目，将一台电脑与一台相机连接起来，让电脑“描述它看到了什么”。因此，它开始寻求解决“视觉输入”问题，这在当时被认为比机器中的认知智能问题更容易。迄今为止，这个问题仍然困扰着专家们。

简化计算机视觉(CV)可以描述为:

*   机器语言、深度学习算法的使用
*   分析、推断、解释和理解图像或视频的内容
*   获取信息。

计算机视觉旨在机器中模仿人类的视觉能力。但是人类的视觉是一个非常复杂的现象。人类可以很容易地描述他们看过的图像或视频，甚至是一次。

他们可以区分图像和视频中的各种颜色、阴影、深度、物体、人物、背景，并可以详细总结其背景、情感、内容和视觉效果。

## **挑战**

对于实现类似人类视觉能力的机器来说，它应该能够感知颜色、深度和距离，识别模式，检测、识别和分类对象，以处理信息并采取行动或决策。为了做到这一点，计算机视觉借鉴了众多跨学科的科学。从生物科学的角度来看，计算机视觉旨在为人类视觉系统提供计算模型。

从工程的角度来看，计算机视觉旨在建立自主系统来执行人类视觉系统可以执行的一些任务，甚至在许多情况下超越人类视觉系统。大多数视觉任务都与从图像或视频等时序 2D 数据中提取 3D 和历史上下文信息有关。

计算机视觉经常与图像处理混淆，但两者是不同的。

*   图像处理是指通过降噪、亮度/对比度校准等步骤创建、重新创建、增强/改变图像质量，以创建另一幅图像。
*   另一方面，计算机视觉专注于解释图像，处理图像或视频的内容和视觉元素，以收集、推断和解释关于它们的信息，用于决策。

## **核心概念**

计算机视觉关注两个核心概念:

*   首先，理解图像或视频场景中的 3 D 几何结构，以收集关于图像或场景中所描绘的环境的 3D 结构的信息。
*   第二，捕捉语义信息，如我们能否像人类一样识别图像中的物体、人物或活动，并理解不同场景组件的结构和关系？

据华盛顿大学的 Steven Seitz 教授所说，计算机视觉的目标是从数字图像中计算三维世界的属性。该领域的问题包括识别环境的 3D 形状，确定事物如何移动，以及识别熟悉的人和物体，所有这些都是通过图像和视频的分析来实现的。

计算机视觉侧重于从图像中恢复世界的三维结构，以用于全场景理解。

来自加州大学伯克利分校的 Jitendra Malik 描述了计算机视觉的关键三个 R——重建、识别、重组。计算机视觉跨越了特征检测、图像分割、运动估计、图像拼接、3D 形状重建和对象识别。

## **目前为止的旅程**

计算机视觉的最初工作围绕着使用神经网络的边缘检测，并能够区分简单的形状，如圆形和方形。到 1970 年，光学字符识别(OCR)能够识别手写笔记和打字备忘录。

过去十年见证了数据的激增，带摄像头的移动电话以前所未有的速度生成图像。查看以下几个数字:

*   仅在过去的两年里，世界上 90%的数据都是由
*   现在每天都会产生令人难以置信的 2.5 万亿字节的数据。
*   预计 2017 年拍摄的照片数量将达到惊人的 1.2 万亿张，年复合增长率为 9%

过去十年还见证了硬件、软件(摄像头、传感器、物联网)的快速技术进步，内置摄像头的手机的出现，更好的网络连接，社交媒体的广泛采用。这些发展汇聚成对不断增长的图像和视频数据集的轻松访问。

更高的数据计算处理能力和卷积神经网络等新算法有助于机器识别照片和视频中的特定人。因此，计算机视觉在诸如面部识别、医学应用的 x 射线研究等领域发展迅速。

## **训练模型**

为了训练这个模型，计算机被输入了数以千计的相关图像。该系统自行学习以区分图像的不同特征，然后基于这些学习来解释测试数据或实时数据。使用深度学习神经网络，计算机可以区分图像的许多不同部分，并识别边缘。然后，他们通过深层网络层使用过滤和一系列操作来建模子组件，然后将这些片段放在一起，形成所有部分，以获得完整的图像。
