VueJS+CubeJS

Cube.js 是一个开源分析框架，允许创建一个包含高性能数据的仪表板。Cube.js 向其他开发者开放，你可以在 [*这里*](https://github.com/cube-js/cube.js/blob/master/CONTRIBUTING.md) *找到如何贡献的指南。*

要使用 Cube.js，您需要在您的 PC 上本地安装它，然后设置后端以允许与前端交互。
