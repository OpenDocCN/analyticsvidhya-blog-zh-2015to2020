# 在 Python 中使用日期时间

> 原文：<https://medium.com/analytics-vidhya/working-with-date-time-in-python-157793e168f9?source=collection_archive---------9----------------------->

处理日期和时间是一项重要的技能，因为许多数据都包含日期/时间信息，例如:

*   带有日期和/或时间的天气数据。
*   计算机日志，带有每个事件的时间戳。
*   包含日期/时间范围的销售数据。

处理日期/时间数据通常要复杂得多:

*   在有复合日期格式的地方，比如*2020 年 1 月 1 日*，分隔每个组成值并将其转换为数字形式是很麻烦的。
