> SQL 基础

参考:[https://www.freepik.com/](https://www.freepik.com/)

SQL 代表**结构化查询语言**，是一种管理和与数据库通信的语言。例如，它用于数据库创建、删除、通过获取行来更新行、修改行等。SQL 语句用于更新数据库数据或从数据库检索数据等任务。
