搜索是 web 浏览中非常常见的场景。当然你可以通过数据库(SQL 或者 noSQL)来搜索东西，但是有一个工具可以缓解你的数据库的请求压力。那就是 Elasticsearch(ES)，一个开源的搜索引擎，支持分布式搜索和丰富的 API。而且，ES 通过 Json 操作数据，对开发者更友好。
