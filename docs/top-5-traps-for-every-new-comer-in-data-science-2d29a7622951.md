## 数据科学初学者指南

## 随着数据科学的岗位数量和技能需求的增加，也有很多陷阱准备把新人拉进去。本文试图启发人们不要陷入其中。

如今，我们见证了数据科学领域的飞速发展。非常迅速！我们看到很多人进入这个空间。与此同时，全球对这些人的需求也在增加。

当前的疫情和世界各地的封锁为有上进心的人追求这一令人惊叹的职业开辟了道路。互联网上充斥着多种资源，可以从整体上学习几乎任何与人工智能相关的东西。大部分是免费的，也有付费的。

在这篇文章中，我们将主要关注每个新手都很容易犯的常见错误。这将完全是我的个人经历，也是我在过去几个月里观察到的人们的所作所为。

## 01.在话题间跳跃

![](img/36e8def0371363db2b8c1e58592f2916.png)

照片由[凯勒·伍兹](https://unsplash.com/@caleb_woods?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄

> 成为一个万事通是件好事。但是什么都不擅长是最糟糕的。

如今，我们看到许多人在 LinkedIn 上分享他们的日常更新和学习。数据科学领域的相对新手被这些吸引人的结果高度吸引，并倾向于跟踪该主题。

例如，如果 Alice 看到了一个与 OpenCV 相关的帖子，她就会被吸引，并开始学习计算机视觉。几天后(有时就在第二天)，在看了另一个与 BERT 相关的帖子后，她开始关注 NLP 讲座。

如果他们继续这样做，他们将不断地在多个主题之间跳跃，永远学不到任何东西。一个人不可能精通任何一个话题。但是一个人应该能够理解一些基本的东西。

把基础学全，明白核心概念是什么。首先，追随任何一个让你感兴趣和着迷的领域。然后你会有勇气和心态在那个特定的领域打下坚实的基础。然后继续下一个。

## 02.相信付费课程更优越

![](img/73cb76809d99fb7aca1f59e00b6ff42f.png)

照片由 [NeONBRAND](https://unsplash.com/@neonbrand?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

这是一个非常常见的陷阱，目标是进入数据科学的新生、学生和其他领域的专业人员。在线付费课程的数量与日俱增。他们建立了一个吸引人的网站，并在课程中加入了许多流行词汇，而且收费很高。有的甚至说课程结束后提供 2 到 3 个月的实习。

来吧伙计们。我们生活在 2020 年。互联网上充斥着信息。只要躺在床上花些时间搜索，你就有大量的信息来学习几乎任何东西。无论你喜欢哪种格式(比如视频、音频或文本)，互联网上都有可用的资源。它们是完全免费的。

您可以在许多著名的文章中找到端到端学习数据科学的资源(理论基础、基本编码示例、数据收集、模型选择、模型构建和模型部署)。如果你想要一个可视化的格式，你有许多 YouTube 频道教你一切。有一些渠道，他们张贴视频，简单地解释艺术研究论文的状态。

即使你想注册一些在线课程，在许多知名的 MOOC 平台，如 Coursera、EDX 等，也有许多知名大学免费提供的精彩课程。

我们可以目睹许多人开始这些付费的在线和辅导课程，只是为了让人们变得愚蠢和赚钱。千万别上当。我们生活在信息时代，这里不缺乏信息。

## 03.只是盯着屏幕

![](img/99682a83564635b7c04a923f59de7155.png)

[思远](https://unsplash.com/@jsycra?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍照

在注册了一些著名的在线课程后，大多数人只是盯着屏幕，直到讲座结束，并在结束时完成测验。完成测验后，他们自豪地在 LinkedIn 上分享他们的证书。但这不会完成目的。证书永远不能证明你已经掌握了那项技能或技术。

如果讲座更多的是一个理论，你会觉得它很容易理解，对你来说是小菜一碟。但事实并非如此。你很快就会忘记事情。你将失去几乎所有你花在看讲座上的时间的价值。手写或数字格式的笔记总是更好。把它记在脑子里一段时间会很有帮助，如果你以后需要的话，它也是一个有用的复习资源。

另一方面，如果讲座主要由数学方程或代码组成，那么在任何 IDE 或笔记本中使用它们总是更好。大多数代码语法看起来很简单，但可能包含一些不太引人注目的部分，这可能会导致一些错误。只有和他们一起工作，你才能理解其用法的长度和广度。看的时候编码看起来简单直观。但是现实只有你自己去努力的时候才能感受到。

## 04.仅靠 Python 是救不了你的

![](img/6ce844f9289e9eb7b7166a3aaaf53162.png)

在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上由 [Toa Heftiba](https://unsplash.com/@heftiba?utm_source=medium&utm_medium=referral) 拍摄的照片

众所周知，python 仍然是数据科学应用的王者。它被广泛采用，广为人知，有一个很大的帮助社区。与此同时，如果你想在这个领域有所作为，你也应该关注不同市场中的其他一些利基参与者。

r 也是 python 的另一个广为人知的竞争对手。它广泛应用于学术机构和研究实验室。由于 R 中的图形库和其他一些图形库是基于设计和可视化原则构建的，所以使用起来非常直观。另一个原因是它是专门为统计分析而构建的。所以它非常强大，在小众领域几乎无人能敌。

SAS 是 python 的另一个竞争对手。与 Python 和 R 不同，SAS 不是开源的，可以在企业版中使用。但是，许多一流的跨国公司仍然使用它，因为它的可靠性、速度和专门的支持。在 R 和 python 中，包括你和我在内的任何人都可以发布包。有时候，如果一个特定的包被更新了，那么构建在这个包之上的其他包也必须被更新。所有这些问题都可能导致混乱和缺乏可靠性。同时，在 SAS 的情况下，一切都由 SAS 团队完成和维护。SAS 也是数据科学的主要参与者之一。

除此之外，还有一些其他工具，如电子表格、Java、Scala 等，可以在某些情况下拯救你。SQL 也是数据科学中非常受欢迎的技能。如今，它已经成为与任何其他编程语言一起使用的基本质量和要求。

## 05.为公司带来价值

![](img/7e0e490fc5d07edd2fc9c3b27abc3784.png)

照片由[奥斯汀·迪斯特尔](https://unsplash.com/@austindistel?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

现在广泛流行的一个常见误解是，数据科学就是建立复杂的机器学习模型和深度学习模型，在云 GPU 上花费数百美元。但实际上，并非如此。数据科学家的最终目标是为公司带来价值。无论他/她可能使用什么模型或架构，最终模型应该满足其目的。

从数据中提取的见解应该有助于公司做出明智的决策，以实现计划目标。甚至可能出现不需要使用任何机器学习方法的情况。简单地使用基于规则的模型可以有效且高效地解决问题。那些刚刚尝试探索这个领域的人应该更多地关注有价值的见解和商业决策，而不是关注模型的复杂性。敏锐的商业头脑也是让你与众不同的重要品质。
