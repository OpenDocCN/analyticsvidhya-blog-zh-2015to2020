安装 pyodbc 需要什么？

**为什么需要 PyODBC？**

通过 Python 连接到任何数据库。(每个数据库都有特定的驱动程序，pyodbc 是一个公共库)

如果您的系统具备所有的先决条件，那么使用 pip 库进行安装是非常简单的。

如果忘记了开发库，在 Centos / RHEL 上安装 pyodbc 可能会很困难。
