今天我们将讨论性能指标，这次是回归指标。在我之前的 博客 中，我们已经讨论过分类指标，这次是回归。

我们将讨论 5 个最广泛使用的回归指标:

让我们先了解一件事，那就是分类和回归指标之间的区别，为什么我们需要两种不同的指标来度量我们的模型。

第一个关键区别是，顾名思义，分类将类作为输出，这可以理解为我们只有很少的数据类别，比如类 1-10，那么我们的输出值将是 1-10 之间的任何数字。因此，如果模型输出与我的实际输出匹配，那么结果通过，否则失败，没有其他条件，你可以是正确的或不正确的。虽然这不是回归的情况，但在回归中，我的模型输出一个连续的数字，没有离散值，它是连续的，例如我们的模型试图预测人的身高，我们知道我们不能将身高变量分类为 160 厘米、170 厘米等。它是连续的，因此在这种情况下，我们考虑我们的模型与实际值有多接近，“有多接近”的概念产生了损失项。 用适当的科学或统计符号来表示，我们的模型在预测一个数据点的价值时会产生什么样的损失。 比方说，对于某个数据点，预计身高为 167 厘米，而实际数据点的实际值为 163 厘米，那么我们的模型就出现了+5 厘米的误差。在这种情况下，现在我只针对 1 个数据点，想象一下如何测量整个数据集？

让我们记住一件事，什么是错误？

与实际值的任何偏差都是误差，

误差 = Y(实际)— Y(预测)

记住这一点，我们已经了解了指标的要求，让我们深入研究一下我们可以用来了解模型性能的方法。

1。均方误差

让我们试着分解这个名字，它表示平均值，平方，误差。从上面的解释中我们知道什么是误差，我们知道什么是平方，所以我们平方误差，然后我们知道什么是平均值，所以我们取所有被平方和相加的误差的平均值。

第一个问题是为什么我们要做一个正方形？为什么我们不能直接接受错误？

让我们再举一个身高的例子，我的模型预测的是 167 厘米，而我的实际值是 163 厘米，所以偏差是+5 厘米，现在让我们再举一个例子，我的预测身高是 158 厘米，而我的实际身高是 163 厘米，现在我的模型出现了-5 厘米的误差。

现在让我们找出 2 个点的平均误差，因此计算状态为[+5 + (-5)]/2 = 0

这说明我的模型有 0 误差，但是真的是这样吗？没有对吗？因此，为了避免这种问题，我们必须采取平方，以消除错误的迹象。

现在的问题是，如果我们已经有了 MSE，为什么我们还需要 RMSE？

让我们试着用例子来理解它，以上面 2 个数据点的例子为例，计算它们的 MSE 和 RMSE，

MSE = [(5)2 + (-5)2]/2 = 50/2 = 25

RMSE = Sqrt(MSE) = (25)^0.5 = 5

5。r 或决定系数

它是目标变量的 MSE(预测误差)和基线方差的比率，这里基线是我们的 Y 值与平均值的偏差。

该指标帮助我们将当前模型与恒定的基线值(即平均值)进行比较，并告诉我们我们的模型好多少，R2 总是小于 1，无论误差有多大或多小，R2 总是小于 1。
