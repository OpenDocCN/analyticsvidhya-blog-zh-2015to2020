# 为什么我想写这个系列

一个月前我在浏览 PTT BBS 时，注意到一个关于建立一个旨在知名数据集上练习的 Slack 群的帖子。看完帖子，我想我可以在这些著名的数据集上做一些副业。更重要的是，我可以在未来运行机器学习项目时，结识一些朋友，获得一些很棒的特征工程和模型选择的想法。

总而言之，我写这个系列的目的是记录我的解决方案，并获得一些反馈来改进我的解决方案。

# 本系列的目的

本系列将介绍数据集的输入要素和特征。此外，我将展示我的步骤，经历这些数据集和我的模型的性能。
