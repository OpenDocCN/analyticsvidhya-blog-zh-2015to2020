这种病毒危险吗？
为了评估这种新型冠状病毒带来的风险程度，需要了解三个参数:

1.  传染率(Ro)——单个病例中新感染的人数
2.  病死率(CFR) —导致死亡的病例百分比
3.  确定无症状传播是否可能
