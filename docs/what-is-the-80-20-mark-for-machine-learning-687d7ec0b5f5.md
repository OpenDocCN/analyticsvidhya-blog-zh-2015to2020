# 问题

无论我想学哪种技术，我总是试图判断 80/20 标记在哪里。我想花最少的努力(~20%)获得最多的价值(~80%)。这是一个粗略的想法，我可能不会总是准确地击中它，也许因为项目要求迫使我学习更多，或者如果我只是碰巧发现一些超级有趣的东西。然而，如果我能够达到接近 80/20 的分数，那么如果需要的话，我应该能够花额外的时间来学习剩下的概念。这也意味着，我不打算…
