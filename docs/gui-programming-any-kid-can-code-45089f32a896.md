# **什么是 GUI？**

GUI 是指图形用户界面。任何具有可视组件的软件，实际用户可以与该软件进行交互，都属于 GUI。例如，用户使用键盘输入、按下鼠标按钮等。

在游戏中，我们有一个界面，我们可以使用不同的输入来玩它。这就是允许在游戏中互动的图形用户界面。
