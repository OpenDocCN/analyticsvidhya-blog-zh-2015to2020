**简介**

近年来，随着大数据的爆炸式增长，组织和数据科学家对使用非传统数据源进行信息提取有着巨大的需求。研究表明，近 80%的数据以非结构化文本数据的形式存在，因此文本分析是分析聊天记录、社交媒体帖子、评论、新闻等大量可用信息的基础
