应用:

二项式分布通常用于模拟从数量为 N 的总体中抽取替换数量为 n 的样本的成功数量。如果抽样是在没有替换的情况下进行的，则抽取不是独立的，因此得到的分布是超几何分布，而不是二项式分布。然而，对于比 n 大得多的 N ，二项式分布仍然是一个很好的近似，并且被广泛使用。
