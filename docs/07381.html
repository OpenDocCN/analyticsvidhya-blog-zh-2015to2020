数据可视化

数据可视化是信息和数据的图形化表示。通过使用可视化元素，如图表、图形和地图，数据可视化工具提供了一种可访问的方式来查看和理解数据中的趋势、异常值和模式。

Python Matplotlib 。 matplotlib.pyplot 是python编程语言中用于2D图形的绘图库。它可以用于python脚本、shell、web应用服务器和其他图形用户界面工具包。
