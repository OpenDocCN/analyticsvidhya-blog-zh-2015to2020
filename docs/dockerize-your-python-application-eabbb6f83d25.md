*首先，docker 是什么，为什么要用？*

![](img/dec66f92cee521b8e8e7a37cb62382f4.png)

由[王德·古普塔](https://unsplash.com/@dewang?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

Docker 是一个容器化软件，它使我们能够将运行应用程序所需的所有依赖项打包，并将其作为一个单独的包部署在一个松散隔离的环境(称为容器)中。Docker 提供了一种高效的方式来共享我们的应用程序，而不用担心安装依赖项。
