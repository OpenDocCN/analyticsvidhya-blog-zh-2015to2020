3。VPC 对等

什么是 VPC 对等？

在不同的 VPC 之间创建连接，简单来说就是 VPC 对等。它允许跨两个 VPC 网络的内部 IP 连接，无论它们属于同一个项目还是同一个组织。

注意:必须从两端(两个网络)建立 VPC 对等。

4。库伯内特星团

什么是 Kubernetes？

Kubernetes 基本上是一个容器管理工具 ( Docker、Podman、Crio 等)。有些需求可以通过不同的容器来解决。但是如果由于某种原因环境恶化了呢？我们无法手动监控。为此，我们需要一些智能工具。库伯内特就是这样做的。它具有自动缩放功能，还内置了负载平衡器。与之相关的一些术语:

→节点: Kubernetes 通过将容器放入 pod 中在节点上运行来运行您的工作负载。根据群集的不同，节点可以是虚拟机或物理机。每个节点都包含运行由控制平面管理的 p ods 所需的服务。 Kubernetes 是一个主 - 从类型的架构，有一定的组件。

→Pod:Pod 是 Kubernetes 集群的一个节点内的容器及其存储的集合。Kubernetes 中的每个 pod 都被分配了一个集群中唯一的 Pod IP 地址，这允许应用程序使用端口而没有冲突的风险
