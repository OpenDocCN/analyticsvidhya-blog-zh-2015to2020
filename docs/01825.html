谷歌搜索有九个基本操作符:

1-精确短语(" "):当您对要搜索的主题有把握时，此运算符可帮助您获得精确的答案。

举例 :

你的尝试 : 卖法拉利的和尚

Google-Fu : 《卖法拉利的和尚》

2-数字范围(。。):此运算符帮助您在搜索结果中设置一系列数字。

举例 :

你试试 : 曼德拉演讲

2004年纳尔逊·曼德拉的演讲..2007年

3- Exclusion (-):该操作符帮助您排除关键字。

例题 :

你试试 : 黑豹

谷歌-傅 : 黑豹-球队-俱乐部

4-相似词(~):该操作符帮助您获得与实际结果相似的结果，这意味着它给出了与查询相关的答案。

例题 :

你试试 : 房子着火

Google-Fu:~着火的房子

5- Inclusion (+):该运算符有助于您包含关键字，方法是在精确使用该运算符后，为单词赋予优先级。

举例 :

你的尝试 : 明日世界Marshmello演出

Google-Fu : 明日世界+Marshmello表演

6-填写空白(*):当您不确定自己的查询是否合适时，这个操作符可以帮助您获得结果。

例题 :

你的尝试 : 加纳最受欢迎

Google-Fu : 加纳最受欢迎的*

7-多个单词(OR):当你感到困惑时，这个操作符帮助你得到可能的结果。

例题 :

你尝尝 :头尾/ 披萨汉堡

:头还是尾/ 披萨还是汉堡

8- site:这个操作符帮助你只从一个特定的网站或给定的域中搜索网页。

例题 :

你试试 : 从亚马逊网上购买衬衫

Google-Fu : 网站:亚马逊衬衫

9- filetype:这个操作符帮助你得到pdf格式的答案。移动，。doc，。xls和许多其他多重扩展。

例题 :

你试试 : 按州人口

Google-Fu : 各州人口文件类型:xls

20种含义和技术可以用来改进你强大的谷歌搜索

我给Google-Fu列出了所有重要的提示和技巧，因为有数百种 搜索查询操作符 和技巧。所以为了节省时间，我把它们都剪短成几个，这样就可以很容易地用于日常活动，也可以让你养成习惯。

1- site:它告诉Google在特定的web域中搜索页面。

例子 : 网站:ebay.com/gadgets

专业提示:“+www . ebay .+com”——(告诉你你的网站有多少页面被谷歌索引)

2- intitle:它告诉Google在给定的词内搜索结果。

示例 : intitle:Oneplus

3- inurl:它告诉Google在结果中搜索url中的某个单词。

例子 : inurl:JBL耳机

4- intext:它告诉Google在任何可用的内容中搜索word，尽管你可以使用其他操作符。

举例 : 正文:切尔诺贝利

5-金融锚:它告诉谷歌搜索结果的锚文本与入站链接。

例子 : 金融家:谷歌Chromebook

6- allintitle:当你想了解竞争对手时，它告诉谷歌搜索一个产品或一个利基市场的结果。

示例:all title:chrome book

7- allinurl:它告诉Google搜索url中包含的所有单词的结果。

例子 : allinurl:谷歌chromebook

8-all text:它告诉Google搜索包含网页中所有单词的结果。

举例:allin text:Google chrome book

9- filetype:它告诉Google以查询中给定的任何格式提供结果。

示例 : 文件类型:pdf人口州美国

10- ext:类似于Filetype

示例 : ext:pdf人口州美国

11- daterange:它告诉Google搜索给定日期范围或年份的结果。

例子 : 迈克尔·杰克逊1995..2005年

12日期:类似于日期范围

例子 : 迈克尔·杰克逊1995–2005

注意:尽管结果不同。

13- weather:它告诉Google搜索给定地点或邮政编码的天气预报。

举例 : 天气:84180 或天气:德国

14- Around(x):大多数情况下，它告诉谷歌将搜索结果限制在包含搜索单词的页面内，这些搜索单词之间的距离不超过X个单词。x只是定义了单词的范围或数量。

举例 : 谷歌富绕⑤手法

15- Stocks:它告诉Google搜索关于股票的结果。

举例 : 股票:facebook

16- map:它告诉Google搜索特定地点的结果。

举例 : 地图:巴塞罗那

17- movie:它告诉Google搜索电影的结果。

举例 : 电影:钢铁侠或电影:【zip code】

18- convert:它告诉Google转换货币和单位。

例 : 换算:美元..inr 或换算:吨..kg

19- source:它告诉Google查找一个主题或一篇文章的来源。

例子 : 来源:如何成为一名谷歌人

20- Hastags:它告诉谷歌去寻找被某人标记的文章、照片、博客。

例子:#友谊目标永远的朋友
