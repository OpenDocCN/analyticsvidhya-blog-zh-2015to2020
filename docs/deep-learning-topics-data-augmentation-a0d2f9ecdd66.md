# 为什么我们需要数据增强？

*   **数据不足**。一些模型可能需要大量我们目前没有的数据。我们可以使用数据增强来为我们创造更多的数据。
*   **过拟合**。如果模型过拟合，我们可能希望在数据中捕捉更多的不变性。
