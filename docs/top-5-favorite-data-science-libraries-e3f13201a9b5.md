# **5。张量流**

TensorFlow 由谷歌大脑团队创建，是一个用于数值计算的开源库，使机器学习变得更容易、更快。帮助开发人员在不影响速度或性能的情况下构建和训练最先进的模型，还可以使用数据流图创建大规模神经网络。它可以处理各种任务，如语音识别和对象识别等。

## 主要特点:

*   有效地处理涉及多维数组的数学表达式。
*   深度神经网络和机器学习任务的最佳支持。
*   相同的代码可以在两种 GPU/CPU 架构上执行。
*   跨各种机器和大型数据集的高可扩展性计算。
*   用于基于文本的应用、视频检测、语音和图像识别。

要了解更多关于 TensorFlow 的信息，请访问以下链接:

[https://www.tensorflow.org/tutorials/](https://www.tensorflow.org/tutorials/)

# **4。Keras**

由 Francois Chollet 开发的 Keras 是一个开源库，为神经网络提供了 Python 接口。运行在 TensorFlow、Theano、Cognitive Toolkit (CNTK)等库之上。利用多种优化技术使高级神经网络 API 变得简单而高效。

## 主要特点:

*   简单、一致且可扩展的 API。
*   支持多平台和后端。
*   一个在 CPU 和 GPU 上运行的用户友好的框架。
*   最小的结构，容易实现的结果没有任何失败。

要了解更多关于 Keras 的信息，请查看[https://keras.io/](https://keras.io/)

# 3.熊猫

由 Wes McKinney 开发的 Pandas 是一个高级数据操作工具，它功能强大、灵活且易于使用，是用于操作数字表和时间序列的开源数据分析和操作工具。Pandas 允许将数据结构转换为数据框对象，处理丢失的数据，从数据框中添加/删除列，也用于绘制直方图或箱线图等。

## 主要特点:

*   由高级数据结构和操作工具组成。
*   允许您创建函数并在一系列数据上运行它。
*   丰富的功能让您可以自由处理丢失的数据。
*   帮助读取不同文件格式的数据。
*   对数据争论和清理有用。
*   支持时间序列功能，如日期范围生成、移动窗口和日期移动。

想在 10 分钟内了解更多关于熊猫的信息，请看这个—

[https://pandas . pydata . org/pandas-docs/stable/user _ guide/10min . html](https://pandas.pydata.org/pandas-docs/stable/user_guide/10min.html)

# 2.Matplotlib

Matplotlib 是我最喜欢的，也是用于数据可视化的最流行的包之一。它还提供了一个面向对象的 API，有助于在应用程序中嵌入绘图。我不会说它很容易使用，但如果你为基本图表和散点图保存一些最常用的代码块，也就是说，很容易更快地创建图表。

# 1.Scikit 学习

由 David Cournapeau 开发的 Google 代码之夏项目。Scikit Learn 是一个免费的 python 机器学习库。它具有朴素贝叶斯、梯度推进、随机森林、SVM 等算法，还支持 NumPy、SciPy 等数字和科学库。

特点:

*   聚类有助于对 K-means 等未标记数据进行分组。
*   交叉验证用于评估监督模型对未知数据的性能。
*   集成方法能够结合多个监督模型的结果。
*   特征选择用于识别用于创建更好模型的重要属性。
*   超参数调整提高了算法的性能。

在这里[https://scikit-learn.org/stable/](https://scikit-learn.org/stable/)进行深入探索
