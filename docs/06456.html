这段代码将简单地 映射 我们在顶部声明的 imgList 中的所有图像到一个容器中。变量的值取决于传递给列表的图像数量。该功能。toList( ) 用于映射列表项。您可以通过使用carouse options()小部件来添加转盘选项。

void add() {

    _n++;
    _amt = _amt + 10;
  });
}

声明一个函数 add ( ) ，当 + 图标被按下时，我们将调用这个函数。

void minus() {

}
