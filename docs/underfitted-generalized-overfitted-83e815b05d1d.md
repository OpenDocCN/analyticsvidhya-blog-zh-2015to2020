# Uunder fitted:

模型可能非常不适合训练和测试数据(高偏差和低方差)-上图中最左侧的图表。这就是所谓的**吃不饱**。

# 过度装配:

模型可以很好地拟合训练数据，而很差地拟合测试数据。(低偏差和高方差)—上图中最右边的图。这就是所谓的**过度劳累。**

# 一般化:

一个理想的模型应该是低偏差和低方差的。
