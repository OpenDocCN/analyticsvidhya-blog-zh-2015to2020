# **总结**

1.  *args 和**kwargs 在向函数传递可变数量的参数时非常方便。
2.  *args 用于传递位置参数。
3.  * * kwargs 用于传递关键字参数
4.  传递不同类型的参数时，顺序应该是标准参数、位置参数(*args)和关键字参数(**kwargs)。
