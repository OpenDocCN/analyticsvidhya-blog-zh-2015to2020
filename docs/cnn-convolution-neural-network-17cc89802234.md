## 深度学习

# 为什么是 CNN？

*   在全连接神经网络中，给定层中的任何神经元都与前一层中的所有神经元完全连接，因此它由许多参数组成，因为它们更容易过拟合。
*   由于有一个巨大的神经元链，该模型可能会遇到梯度消失的问题。
*   我们的目标是得到更好的优化算法，更好的激活函数，更好的初始化方法，更好的正则化。
