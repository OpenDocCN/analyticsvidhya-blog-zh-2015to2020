# **A/B 测试**

A/B 测试是对两组进行的实验，以确定哪一种处理更好。在实验中，**治疗组**的受试者接受特定的治疗，而**对照组**的受试者不接受任何治疗。通常，受试者被随机分配**以消除取样偏差。**置信区间**和**测试统计**是一些用于衡量治疗效果的指标。**
