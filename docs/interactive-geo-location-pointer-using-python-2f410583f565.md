# 保存地图

这样生成的地图可以被保存以供将来参考，

> variablename . save(" filename . html ")

这将创建一个 HTML 文件，其中包含之前生成的地图。

如果需要修改，可以进一步修改。

**举例:**

> map1.save("map1.html ")
