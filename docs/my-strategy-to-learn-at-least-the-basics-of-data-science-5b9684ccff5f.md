学习数据科学可能会令人望而生畏，尤其是当有数千门课程可供选择时。最初，我认为数据科学就是编码。我会跟着 YouTube 上的一些教程，然后嘣…我是一名数据科学家。然而，数据科学不仅仅是编码。它还包括数学、统计学、一些商业知识等等。所以，你真的需要时间去学习数据科学的基础知识。

尽管学习数据科学很难，但如果你一步一步来，并高效地学习，它会变得很容易。为了高效地学习数据科学，我设计了一个计划来帮助我至少学习数据科学的基础知识。您可以使用相同的计划来帮助您的数据科学职业生涯，看看它是否对您有用。

## **第一步:学习数学**

![](img/f5852c78b26c0707b7762a821284198f.png)

照片由 [Antoine Dautry](https://unsplash.com/@antoine1003?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄

对于数学，你需要知道统计学、线性代数和微积分。这些科目的所有基础都可以从可汗学院免费学到！Khan 在向初学者解释所有主要的数学概念方面做得很好。然而，你不需要马上学会每一门数学课，因为它会让你不知所措。最初，我会建议学习统计学，因为对于数据科学家来说，分析图表和趋势是至关重要的。学完统计学后，你可以继续学习线性代数，然后是微积分。然而，你可以选择任何你喜欢的数学科目并开始学习。这里是每个数学科目的链接

统计:[https://www.khanacademy.org/math/statistics-probability](https://www.khanacademy.org/math/statistics-probability)

线性代数:[https://www.khanacademy.org/math/linear-algebra](https://www.khanacademy.org/math/linear-algebra)

微积分:[https://www.khanacademy.org/math/differential-calculus](https://www.khanacademy.org/math/differential-calculus)

如果你知道代数 1 的基础知识，统计将会很容易。对于线性代数，你应该知道代数 2 的基础。对于微积分，你应该知道微积分的基础知识和代数 2。

如果你对代数 1、代数 2 和微积分先修课程不熟悉，我强烈建议你先学习这些。同样，你可以在可汗学院学习这些课程。

然而，值得花时间去学习所有这些数学科目吗？当然啦！下面解释了这些数学科目将如何应用到您的数据科学职业生涯中。

**统计:**作为一名数据科学家，你应该知道你的统计数据**。**统计是分析图表和趋势的重要工具。例如，您可以收集一家公司的 XYZ 产品的一些数据，绘制数据图表，并通过使用标准差、平均值、中值和/或众数分析图表来查看该产品是否受客户欢迎。

**线性代数:** NumPy 是数据科学家使用的一个流行模块，它要求你理解向量和矩阵。然而，在数据科学中，线性代数不仅仅是向量和矩阵。

**微积分:**微积分对于计算导数、曲线斜率和/或梯度下降很重要。它主要用于关键的机器学习算法，因此，如果你想从数据科学过渡到 ML 或成为一名成功的数据科学，了解微积分是很好的。

## 步骤 2:学习如何编码

数据科学中最常用的编程语言是 Python、SQL 和 r。几乎所有开发人员都建议从 Python 开始，因为它是一种通用语言。然而，我个人建议一起学习 SQL 和 Python，因为这两种语言在您的数据科学职业生涯中是齐头并进的。您将使用 Python 库，例如 NumPy、Pandas 和/或 MatplotLib 来分析和可视化数据。学习 SQL 对于了解如何与数据库交互和创建表格至关重要，这是数据科学家的主要工作之一。为了熟悉 Python，[自动化枯燥的东西](https://automatetheboringstuff.com/)是一本很棒的免费在线书籍。对于 SQL，由 Mike Dane 教授的本 [SQL 课程非常适合让您开始学习 SQL 和数据库的基础知识。](https://www.mikedane.com/databases/sql/)

现在不要担心学习 R，而是专注于 Python 和 R。然而，一旦你熟悉 Python 和 SQL，学习 R 将是一个好主意。

## **第三步:如何学习**

考虑到你有这么多事情要做，你怎么能在空闲时间把它们都挤出来呢？学习数据科学，不需要在一天内学会所有的东西。为了学点东西，每天选择一个不同的主题，花 30-60 分钟学习这个主题。比如今天你可以学一门数学科目，明天你可以学 Python，后天你可以学一些 SQL。为了记住所有的东西，一定要每周复习一次。例如，如果你有一天学习了 Python 的概念，那么下次你坐下来学习 Python 的时候，复习到目前为止你在 Python 中学到的所有东西。

# 结论

学习数据科学是一场马拉松，而不是短跑，因此，请慢慢来。没有人能在一个月或六个月内成为数据科学家，除非你有数学或计算机科学学位。但是，有一个扎实的计划学习数据科学，会让你轻松地学会至少数据科学的基础知识。即使我不擅长制定循序渐进的计划，你也可以试试我的计划，看看对你是否有效。
