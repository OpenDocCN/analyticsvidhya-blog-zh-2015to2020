迭代意味着逐个遍历元素。

由于我们在python (numpy)中使用多维数组，所以我们可以使用进行循环。

迭代一维:

对于一维数组，迭代一个接一个地进行。

import numpy as np
arr=np.array([1,2,3])
print("Original array:",arr)print("After iteration:")
for x in arr:
    print(x)
