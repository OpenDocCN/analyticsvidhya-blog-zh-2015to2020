# 张量流:

一个常用于 ML 任务的软件是 TF。它被广泛采用，因为它提供了一个接口来表达常见的 ML 算法和模型的可执行代码。在 TF 中创建的模型可以移植到异构系统中，从移动电话到分布式服务器，几乎不需要做任何改变。TF 由 Google 创建并维护，在公司内部用于 ML 目的。

![](img/6ed7dc7981c5a81cc4348ce9e85e73e3.png)

TensorFlow 是一个在大规模和异构环境中运行的机器学习系统。它使用数据流图来表示计算、共享状态和改变该状态的操作。它将数据流图的节点映射到集群中的许多机器上，以及一台机器内的多个计算设备上，包括多核 CPU、通用 GPU 和定制设计的称为张量处理单元(TPU)的 ASICs。

# TensorFlow Lite:

在设计期间，Google 开发了能够在异构系统上运行的 TF，包括移动设备。这是因为当计算可以在设备上执行时，在设备和数据中心之间来回发送数据的问题。TFM 使开发人员能够创建交互式应用程序，而不需要 ML 计算的网络往返延迟。

由于 ML 任务在计算上是昂贵的，模型优化被用来提高性能。就随机存取存储器(RAM)大小和 CPU 速度而言，TFM 的最低硬件要求较低，并且主要瓶颈是计算的计算速度，因为移动应用的期望等待时间较低。

![](img/fb8bb42af7187e9b2451a9eb74313cae.png)

TensorFlow Lite (TFL)是 TFM 的演进，已经支持在移动和嵌入式设备上部署。因为存在将 ML 结合到移动应用中的趋势，并且因为用户在照相机和语音方面对他们的移动应用有更高的期望。TFL 中包括的一些优化是通过硅层的硬件加速、Android 神经网络 API 等框架和 MobileNets 和 SqueezeNet 等移动优化的神经网络。TF 训练的模型由 TF 自动转换为 TF 模型。

# CUDA 和 cuDNN:

CUDA 由 NVIDIA 开发，用于在支持 CUDA 的 GPU 上创建并行计算接口。该平台的功能是作为通用计算的软件层，开发人员可以利用它来执行虚拟指令，并支持许多编程语言。

![](img/f6fce2ea25054cb25335892a16147e1b.png)![](img/99bc4de2742040afd3e14194b9832c4e.png)

NVIDIA CUDA 深度神经网络库(cuDNN)支持 GPU 加速的深度神经网络的训练和推理，用于 ML 中的常见例程和操作。由于 ML 严重依赖于计算能力的可及性，这在训练更大的网络或训练图像等高维数据时至关重要。cuDNN 库为低级 GPU 性能调优提供了强大的支持，使 ML 开发人员能够专注于网络的实现。cuDNN 支持并加速 TensorFlow 中的操作。

# 咖啡:

Caffe 为培训、测试、微调和部署模型提供了完整的工具包，并为所有这些任务提供了记录良好的示例。因此，对于寻求进入最先进的机器学习的研究人员和其他开发人员来说，这是一个理想的起点。同时，它可能是这些算法最快的可用实现，使其立即可用于工业部署。

![](img/29db6c11e768eeaee4eab2a0945046d7.png)

在 Caffe 中，多媒体科学家和从业者拥有一个有序且可扩展的工具包，用于最先进的深度学习算法，并提供开箱即用的参考模型。快速 CUDA 代码和 GPU 计算通过在单个 K40 或 Titan GPU 上实现每天超过 4000 万张图像的处理速度来满足行业需求。相同的模型可以在各种硬件上以 CPU 或 GPU 模式运行:Caffe 将表示从实际实现中分离出来，异构平台之间的无缝切换促进了开发和部署。Caffe 甚至可以在云中运行。

# Theano:

Theano 允许用户象征性地定义数学表达式，并通过修改配置标志，在 CPU 或 GPU(后者使用 CUDA)上以高度优化的方式编译它们。此外，它可以自动计算复杂表达式的符号微分，忽略计算最终输出不需要的变量，重新使用部分结果以避免冗余计算，应用数学简化，在可能时就地计算操作以最小化存储器使用，以及应用数值稳定性优化以克服或最小化由于硬件近似引起的误差。为了实现这一点，由用户定义的数学表达式被存储为变量和操作的图形，在编译时被修剪和优化。

![](img/86079c0c8e0e0077cdca8ef1f3cbcdb6.png)

Theano 的接口是 Python，这是一种强大而灵活的语言，允许快速原型化，并提供了一种快速而简单的方法来与数据进行交互。Python 的缺点是它的解释器，在很多情况下，就内存使用和速度而言，它都不是执行数学计算的好引擎。Theano 通过利用 Python 语言的紧凑性和延展性，并将它们与快速和优化的计算引擎相结合，克服了这一限制。

# Keras:

Keras 是一种高级神经网络 API，用 Python 编写，能够在 TensorFlow、CNTK 或 Theano 之上运行。它的开发重点是支持快速实验。使用 Keras 的优势在于:

![](img/d647195eb517ede451c77b4ca3895cb4.png)

*   允许简单快速的原型制作(通过用户友好性、模块化和可扩展性)
*   支持卷积网络和递归网络，以及两者的组合
*   在 CPU 和 GPU 上无缝运行

# PyTorch:

PyTorch 是一个 Python 包，它提供了两个高级特性:

*   具有强大 GPU 加速的张量计算(如 NumPy)
*   建立在基于磁带的自动签名系统上的深度神经网络
