使用字符串

作为数据科学或一般编码领域的初学者，我很难处理字符串。在Python中读取、操作和匹配字符串对我来说是一个挑战。像区分大小写、行尾字符、括号这样的问题是我最近学会解决的。让我分享一下我学到的帮助我处理字符串操作的东西。

劈弦

我在使用string时学到的第一件事就是拆分它。假设您有一个类似“我的名字是Ed”的字符串split()函数帮助将这个字符串分解成每个单词，并将其存储为一个字符串列表。

string = "我的名字是Ed。"
string _ split = string . split()

Output: string_split= ['My '，' name '，' is '，' Ed ']

现在想象一下，我们只关心字符串中的名字。我们可以这样做:

name = string _ split[len(string _ split)-1]
Output:name = ' Ed '

这是一个很基本的方法。我们将从列表中取出最后一项，并将其存储为“name”。

但是，如果我们有这样的字符串:“我的名字是波特，哈利·波特先生”

在这种情况下，我们总是可以使用上面提到的方法。但这将是解决这个问题的一个非常粗糙的方法。如果我们的名字有多个中间名，它就会失败。在这种情况下，我们将寻找将名字与中间名和姓分开的字符(在这种情况下，它是'.'在“先生”的末尾。

first_name = string_2.split(' . ')[1].lstrip()。split(')[0]
输出: first_name = '哈利'

Split()拆分“.”上的字符串性格。所以我们得到了一个由“.”两边的两个元素组成的列表(['我叫波特，先生'，'哈利波特'])。lstrip()从列表的index [1]元素中移除所有从左开始的前导空格，例如“Harry Potter”。Split()然后进一步将第一个元素除以“”，然后[0]选择第一个被拆分的单词，即“Harry”。

现在想象我们有这样一个字符串“哈利波特(学生)”；姓名(职业)。我们想提取这个人的职业。

string= "哈利波特(学生)"
occupation = string . split('(')[1]。rstrip(')')
输出:【学生】

那很简单对吧！

同样，如果您想要提取名称:

string= "哈利波特(学生)"
name = name _ job . split('(')[0]。rstrip()
输出:《哈利波特》

“案例”研究

我面临的下一个问题是匹配字符串时区分大小写。如果我们有两个字符串“HARRY”和“Harry ”,如果你比较这两个，你的输出显示False，因为python是区分大小写的。如果希望忽略大小写，只需在比较时对每个字符串使用casefold()函数。这个函数告诉python比较字符串中的字母，不考虑它们的大小写。

您还可以使用lower()和upper()这样的函数来改变字符串的大小写。

替换单词

我发现的另一个有趣的特性是replace()。函数将指定字符串中的一个子字符串替换为另一个子字符串。

string= "我叫波特，哈利波特先生"
short = string.replace("，先生，":)
T5】输出:"我叫波特:哈利波特"

python中其他有用的函数有:count()(计算一个字符/字符串在字符串中出现的次数)，startswith()/endswith()(检查字符串是否以字符/字符串结尾)，capitalize()(大写字符串的第一个字母)。

还有许多其他函数可以用来以不同的方式操作、拆分和读取字符串。但是上面提到的函数已经帮助我克服了迄今为止我必须面对的所有与字符串相关的问题。我相信边做边学。所以我期待着通过在我的代码中遇到新问题来学习新的特性和功能。我希望这篇文章对你有所帮助。:)
