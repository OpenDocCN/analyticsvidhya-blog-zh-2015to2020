让我们先玩同样酷的功能-

M = magic(n)返回一个n乘n矩阵，该矩阵由行和列和相等的整数1到n构成。阶n必须是大于或等于3的标量。

sum(A，1):-对的列中的连续元素进行操作，并返回每列总和的行向量。

sum(A，2):-对A的行中的连续元素进行操作，并返回每行总和的列向量。

axis([xmin xmax ymin ymax zmin zmax]):-设置当前轴的x轴、y轴和z轴的限制。v = axis返回包含x轴、y轴和z轴缩放因子的行向量。v有四个或六个分量，分别取决于当前轴是二维的还是三维的。

imagesc(C) :-将C显示为图像。C的每个元素对应于图像中的一个矩形区域。C的元素的值是当前色图中的索引，其确定每个色块的颜色。imagesc(x，y，C)将C显示为图像，并用向量x和y指定x轴和y轴的边界

使用进行绘图的程序，使用:-

for n = 1:16
 subplot(4,4,n)
 ord = n+8;
 m = magic(ord);
 imagesc(m)
 title(num2str(ord))
 axis equal
 axis off
end
