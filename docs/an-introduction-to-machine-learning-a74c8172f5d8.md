# 介绍

学习开阔了我们的视野，增加了我们的知识，并使我们跟上了当今世界。起初，学习仅限于生物。但是在过去的几十年里，人类已经设法创造出了能够学习的机器。创造了机器学习这一术语的亚瑟·塞缪尔将其定义为:“赋予计算机思考的能力，而无需明确编程。”如果未来和过去没有太大的不同，机器可以预测未来。

我们举个例子。我们绘制特定股票或指数的价格与时间线的关系图。我们收集特定时间股票价格的历史数据，并以点的形式绘制在图表上。现在我们画一条穿过图上大多数点的直线。现在我们有了一个线性图。使用这个图表，我们可以估算某一特定时间的股票价格。

**机器学习:**计算机能够在没有明确编程的情况下从经验中学习。

但是有一个问题可能还在我们的脑海中挥之不去？机器到底是怎么学习的？

机器通过在提供的数据集中寻找模式来学习。输入机器的数据越多，输出的准确性就越高。你让机器接触的数据越多，它就越“聪明”。当机器在数据中看到足够多的模式时，它开始基于这些模式进行预测。机器使用旧数据来预测最有可能发生的事情。这就像回顾过去展望未来。

# 机器学习的类型:

## 1]监督学习:

在这种类型的机器学习中，机器学习基于示例输入-输出对将输入映射到输出的函数。该训练数据被标记。有两种监督学习-

**1.a)分类:**

分类问题是当输出变量是一个类别时，例如“红色”和“蓝色”，或者“恶性或良性”。分类用于解决一些问题，例如根据一些参数预测一个人是否患有癌症。这基本上就像是一个是或否的问题。

**1.b)回归:**

一个回归问题是当输出变量是一个实值时，如“卢比”或“重量”。示例—预测特定地区的房地产价格或前面提到的预测[股票](https://www.analyticsvidhya.com/blog/2018/10/predicting-stock-price-machine-learningnd-deep-learning-techniques-python/)市场价格的示例。

## 2]无监督学习:

在这种类型的机器学习中，机器被提供有未被标记的数据。因此，机器的任务是检测数据中的模式并将它们分组。机器可以自己学习。与监督学习不同，机器没有正确的答案。它可以进一步分为以下几类

**2.a)聚类:**

在这个问题中，机器试图在提供的数据中寻找相似之处。如果有一个共同的群体或“集群”，那么算法将它们归类在一起。例子- [Google](https://news.google.com/?hl=en-IN&gl=IN&ceid=IN:en) news 将基于同一新闻的不同文章归为一组等等。

**2.b)联想:**

在这个问题中，机器试图发现描述大部分数据的规则。例如:商店将顾客通常一起购买的产品分类。如果一个人买洋葱和西红柿，他也可能买汉堡肉。

## 3]强化学习:

这背后的想法是，代理将通过与环境的交互来学习。一个婴儿走近壁炉。这是温暖和积极的，宝宝感觉很好(+1)。但是宝宝一碰到火就会烫到自己(-1)。这是人类的方式。强化学习只是一种从行动中学习的计算方法。当机器正确学习时，它会得到奖励。当它学习不正确时就会受到惩罚。例子— [打败了世界上最好的围棋手的机器 AlphaGo](https://deepmind.com/research/alphago/) ，利用强化学习学会了下围棋。

# **当前应用:**

机器学习无处不在。我们只是没有意识到。它用于从我们的电子邮件中过滤垃圾邮件。它用于股票市场，预测市场的未来是牛市还是熊市，并在眨眼之间执行交易。它在医疗保健领域用于疾病的成像和诊断以及收集患者信息，医生可以利用这些信息来提供更好的服务。事实上，任何问题都可以使用机器学习来解决，但前提是我们有足够的数据(并且问题是可解决的)。

# 未来:

机器学习正以极快的速度发展。它比任何人都能更快地获得新的见解和知识，发现趋势。但是，让它变得高效和完美仍然需要大量的人类才能，因为机器还没有足够的技能来确定一个情况的背景。他们还没有掌握推理。所有文件的专家将不得不训练机器，并把它们纳入我们需要改进的过程。机器学习将逐步解决[未来](https://learn.g2.com/future-of-machine-learning)的问题。
