程序大小的增加使得程序更难维护，对于程序员来说可读性和可理解性更差。因此，代码重用是编程的一个重要部分。为了避免重复和使代码可重用，一个大的程序被分解成称为函数的更小的单元。

函数是一组执行特定任务的相关语句。这段代码仅在被调用时运行。
