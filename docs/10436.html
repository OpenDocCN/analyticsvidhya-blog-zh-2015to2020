当您开始数据科学之旅时，感到困惑是很常见的，尤其是因为有大量的资源可以用来学习您想要学习的东西。如果你搜索可用的最佳资源，你将“淹没在黑暗中”,阅读不同的观点、书评、文章，当然还有MOOCs。我明白了，有时Youtube上的推荐，Linkedin上的滚动，Instagram上的帖子，或者一个有效的谷歌搜索结果，会起到一本声誉良好的教科书的作用。

我正在分享一些非传统的资源，比如Youtube视频或从Linkedin、Medium保存的文章，或者超级初学者友好教科书中的特定章节。我还会在每个部分列出所有受欢迎的书籍，这样你就不用再去寻找“最佳资源”了。(一站式解决方案，如果你愿意的话)

成功不是来自“偶尔”做到最好，而是来自“持续”做到更好。

第1部分:学习数据科学编程的资源

计算机编程语言

如果你以前接触过任何计算机语言，我认为你不需要一本全面的python教科书来学习数据科学的python。

在为数据科学学习python时，不要被深入钻研python编程和成为一名全面的python开发人员所淹没，重要的是要专注于学习语法和其他python基础知识，如列表理解、映射、归约、过滤等。让你开始。两个免费的在线资源可以帮你做到这一点，分别是Python like your mean、 Learn python 和 Mode 。

练习Python语法和基本操作，可以利用Pynative Python练习题、练习Python 、 Hackerrank 。

一旦你熟悉了Python基础知识，要学习Python中的数据操作、数据预处理和数据可视化，投资一本完整的书而不依赖在线资源是一个更好的主意。 Python数据科学手册(免费PDF)第2-4章和用于数据分析的Python，由熊猫创始人本人执笔。如果你要从这两本书中选择一本，你的选择应该取决于你对熊猫和熊猫的了解程度。后者更深入高级Numpy和高级熊猫。

Udemy有很好的课程，涵盖了python的基础知识和一些使用python的数据预处理。看看这个。警告:只是一切的基础。

稀有

我最初是通过一个的远程课程学习R的。然而，本课程涵盖了基本概念，如果你是一个完全的初学者，这是非常好的。CodeAcamedy的自由资源做同样的工作。

这个是一个比第一个更强化的课程，涵盖了从基础到教授一些机器学习算法的所有主题。

如果你真的很想深入学习R，你应该考虑投资《数据科学的编程技巧》这本书。这是您需要掌握r语言的唯一一本书。我强调“唯一”是因为大多数统计学书籍都很好地展示了r语言的统计能力。所以当您学习统计学时，无论如何都会看到它。

结构化查询语言

到目前为止，您可能已经听到“数据科学世界”在向您宣传SQL的重要性。

在深入SQL之前，建议先具备 关系数据库系统 的基础知识。

SQL Server初学者教程播放列表 说到SQL概念，是的守护者。如果您完全不熟悉SQL，或者如果您想刷新基础知识并学习一些高级概念，这是一个非常棒的教程(总共150个视频)。(可选零件18–21、29–33)。

你可能会觉得，播放列表的一个缺点是，播放列表的组织不够完善。如果你想细分主题并以更有条理的方式观看，这篇文章可能会帮助你的计划。

SQL面试问答 是以上系列的延续，讲解了最常被问到的面试问题和概念。

大多数人在刚开始的时候很难掌握Windows函数的概念。这篇文章给出了这个概念的直观解释，并且解释了它与按功能分组的不同之处。 Kaggle Advanced SQL 是一个简短的课程，很好地解释了这个概念以及其他一些高级概念，比如查询优化和索引。
