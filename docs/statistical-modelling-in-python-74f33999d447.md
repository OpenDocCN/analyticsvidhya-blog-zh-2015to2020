# 统计建模——概述

> 统计建模让你能够评估、理解和预测数据。

在各种工程领域(机械工程、核工程)、自然灾害管理、质量控制和金融中，统计建模是风险分析和安全的重要组成部分。

在这篇博客中，我将解释统计建模的两个重要主题，即使用统计模型的**线性回归**和 **ANOVA** 。
