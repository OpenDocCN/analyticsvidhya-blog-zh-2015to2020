逻辑图存在病理问题，因为r > 4的值会导致一些负面结果。

对于不同的 r 值，该等式具有不同的行为:

当 0 ≤ r < 1， 种群将在初始条件不变的情况下灭绝。

当 1 ≤ r < 2， 时，尽管存在初始条件，但种群会迅速逼近 (r-1)/r 。

当 2 ≤ r < 3， 最终会达到 (1-r)/r 并围绕该值波动。

当 3 ≤ r < 3.5699， 事情开始变得诡异，3≤r<3.44949对于几乎所有的初始条件，种群都会达到两个值之间的永久振荡。当3.44949≤r<3.54409时，粒子数将出现所谓的倍周期现象，并达到永久振荡量的四个值。而同样当 3.54409 ≤ r 时，种群会分叉到 8，16，32，… 值。两个连续分叉间隔之间的长度接近被称为费根鲍姆常数的 4.669 。

当 r ≤3.5699 时，那么种群将表现出混沌行为，分岔将不再可见，但r中仍有一些区域 再次显示出分岔。

而当 r ≤ 4 时，几乎所有的初始值都会给区间[0，1]留下一些负值。
