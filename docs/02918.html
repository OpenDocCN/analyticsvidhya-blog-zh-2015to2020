λa.a，恒等式组合子

是一个接受输入a并返回输出a的函数。把λa.a想象成类似于f(a) = a的东西，这会让你感觉更舒服。

JavaScript函数I

为了好玩，我们把JS repl提示符改成⬡。在当前目录下创建一个文件repl.js。

var repl = require('repl');repl.start( {
  prompt: '⬡ ',
  replMode: repl.REPL_MODE_STRICT,
  ignoreUndefined: true,
});

使用自定义提示运行节点:

λf.ff，模仿鸟组合子

是一个有趣的功能。它接受一个函数作为输入，并调用传入自身的函数。这是自我应用组合子。

JavaScript函数M

👍 node repl.js
⬡ const I = a => a
⬡ const M = f => f(f)
⬡ M
[Function: M]
⬡ M(I)
[Function: I]

哈斯克尔函数m

不幸的是，我们不能在Haskell中这样做。:-(

参见模仿一只知更鸟获得解释:

ghci >设m z = z z

这是Smullyan的模仿鸟功能。Haskell不允许
函数具有像这样的所谓“无限类型”，尽管它们
可能有用(Haskell提供了其他方法来获得无限类型
会给你的东西)。
