使用Travis CI和AWS Elasticbeanstalk创建Django CI/CD管道

原文：https://medium.com/analytics-vidhya/creating-a-django-ci-cd-pipeline-with-travis-ci-and-aws-elasticbeanstalk-b91bfedd144c?source=collection_archive---------9-----------------------#2020-03-17

本文介绍了使用Github、Travis CI和AWS ElasticBeanstalk为Django应用程序设置CI/CD管道的整个过程

使用的组件:

  • Github :用于版本控制和代码托管
  • Travis CI :针对AWS ElasticBeanstalk的应用构建和部署
  • AWS Elasticbeanstalk :用于托管Django应用程序。
