斯坦福 coreNLP 是基于 java 的。这篇文章是关于它在 jupyter notebook (python)中的实现。斯坦福 coreNLP 可以用来提取多个特征，这些特征可以用来训练任何基于文本的机器学习模型。在大数据上实现可能有点棘手。斯坦福 coreNLP 比 Vader 情感和文本 blob 表现得更好，因为该模型“实际上基于句子结构建立了整个句子的表示，而不是查看单个单词的情感。
