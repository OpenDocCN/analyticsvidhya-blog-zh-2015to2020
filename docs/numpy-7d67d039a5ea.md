Numpy 是 Python 对多维数组的扩展包。它是为科学计算而设计的。它也被称为 ***面向阵列计算*** 。

# **基础知识**

*   它的主要对象是齐次多维数组。
*   数轴称为轴。
*   Numpy 的数组类叫做 ndarray。
*   Numpy 数组不同于标准的 Python 库类数组。
