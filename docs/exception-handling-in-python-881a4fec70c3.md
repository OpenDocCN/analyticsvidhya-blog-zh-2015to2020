> 什么是异常和异常处理？

假设你正在从家到办公室的路上，但是在中途你的车被扎破了，所以你的车会停下来。这种情况被称为异常。现在机械师会来修理你的车，你将到达办公室，修理爆胎的过程被称为异常处理。
