如果您已经创建了一个机器学习(ML)模型，并希望您的朋友尝试一下，最好的方法是将您的模型部署在 flask 服务器中。对于想要部署 ML 模型的初学者来说，Flask 太方便了。Flask 是一个用 python 编写的微型 web 框架。

在本文中，我们将看到如何创建一个 ML 模型来预测印度超级联赛(IPL)第一局的分数，并将其部署在 Flask 中。如果想查看 app，请点击这里。
