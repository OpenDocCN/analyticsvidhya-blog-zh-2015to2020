# 介绍

在机器学习中，支持向量机(SVMs 此外，支持向量网络)是具有相关学习算法的监督学习模型，其分析数据并识别用于分类和回归分析的模式，但主要用于分类分析。它通过在不同类别的数据之间建立一个**超平面**来工作。如果数据不能线性分离，SVM 将尝试将数据投射到一个更高的维度中，用所谓的 ***内核技巧*** 将其分离。
