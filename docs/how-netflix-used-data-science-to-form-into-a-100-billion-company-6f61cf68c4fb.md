# 网飞如何从成堆的数据中获得卓越的用户体验和金钱

虽然数据是一种基本需求，但没有经过适当处理的数据只是一堆无用的信息。

当您处理这些数据以获得有用的见解，从而带来客户满意度和利润时，真正的游戏就开始了。

**网飞使用各种传统的商业智能工具**(如 **Teradata** 和 **MicroStrategy** )并结合现代大数据技术，如 **Hadoop** 、 **Hive** 等。因此，它形成了预先决定用户最有可能观看什么的算法。同样，它也使用了很多开源软件。

> [Justin Ward](https://diginomica.com/forget-orange-netflix-big-data-black) ( **数据科学与工程团队经理，网飞**)说:“大数据对一个数据科学团队来说或多或少都意味着什么。它比你习惯处理的存储空间更大，所以你需要寻找替代方案。这比你以前必须做的处理要多得多。你需要更多的技巧来找到相关性和惊喜。许多可以处理这种分析的工具并不像你期望的那样工作。这是一个新世界，你需要用适合自己的方式去探索它。”

因此，基本上，比拥有数据更重要的是知道如何引导数据，为您的企业和客户带来有利的结果。

# 网飞的艺术品个性化

网飞的个性化推荐系统的主要目的是在正确的时间把最合适的书放在正确的观众面前。

但是推荐的作用并没有到此为止。标题应该足够吸引人，让浏览者点击它，看它能提供什么。这就是艺术品发挥作用的地方。

![](img/ee494ff555c65e4c2055192122ca9ab4.png)

**网飞的艺术品个性化，**来自 [Pexels](https://www.pexels.com/photo/couple-love-sitting-evening-4009401/?utm_content=attributionCopyText&utm_medium=referral&utm_source=pexels) 的 [cottonbro](https://www.pexels.com/@cottonbro?utm_content=attributionCopyText&utm_medium=referral&utm_source=pexels) 拍摄的照片。

艺术作品向观众提供了一种关于标题的视觉证据，从而帮助他们决定是否观看该节目或电影。

网飞不仅仅向 1.83 亿人提供产品。它以 1.83 亿种不同的方式提供产品。

**你看到的艺术品之所以在那里，是因为它包含了你可能会觉得有吸引力的东西。**它可能会突出显示你可能认识的演员的脸，也可能会显示你可能会感兴趣的场景。

**例如，**让我们考虑死亡诗社——一个有几部浪漫电影观看历史的人可能会看到突出伊桑·霍克的艺术作品。同时，一个搜索艺术史或历史电影的人可能会展示洞穴中的一个场景。并且，一个有高中戏剧观看历史的人，可能从教室里看到一个场景。

现在的问题是，他们是如何实现的？通过 **AVA** ，简称**艺术品视觉分析**。

# AVA(艺术品视觉分析):

这是一种算法，可以浏览可用的视频，并选择最适合它们的图像。

AVA 考虑了几件事，比如**场景照明**、**场景中突出显示的内容**、**物体放置**、**背景**等。

****网飞对大量数据进行操作。这些数据涌入几个不同的平台，如[亚马逊 S3](https://www.google.com/aclk?sa=L&ai=DChcSEwiRjaP6kYLrAhUZqpYKHYg5BMwYABAAGgJ0bA&sig=AOD64_22mYxvx6nbgys0ydgIkzYvq-uOeg&q&adurl&ved=2ahUKEwix0Zf6kYLrAhXP6XMBHXkzCQIQ0Qx6BAgdEAE) 、[杜里德](https://druid.apache.org/)、[红移](https://aws.amazon.com/redshift/)等。为了保持互操作性，也就是说，为了在这些不同的平台之间保持信息的顺畅交换和使用，它需要 Metacat。****
