作为一种开源语言，ython 有各种各样的库，使我们能够在几秒钟内完成原本需要几天才能完成的任务。

使用 Python 作为数据科学工具的人都很了解 Matplotlib 和 Seaborn。使用单行代码绘制图形是这些库的特色之一。

如果您想通过绘图来理解您的数据，那么这些一行程序将会派上用场。但是，如果你想你的情节是像样的，简约的，没有任何视觉混乱，你可能要多努力一点。
