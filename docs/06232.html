介绍

二进制决策树是一种supervised机器学习技术，通过对属性进行一系列二进制(是/否)决策来操作。每个决定都有两种可能性。每个决定导致另一个决定，或者导致预测。一棵经过训练的树的例子将有助于巩固这个想法。了解培训的结果后，你将了解培训是如何进行的。 Decision Trees用于Regression和Classification机器学习问题。术语Classification And Regression Tree ( CART)分析是一个总括术语，用来指代这两个过程。
