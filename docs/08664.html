Python 字典是键值对的集合。字典是可变的(可以改变)，无序的，可以被索引(使用键)。尽管事实上字典是可变的，但键不是。在 Python 中，dictionary 属于dict数据类型，是数据结构(内存对象)之一。Python 中的其他数据结构包括列表、元组和集合。
