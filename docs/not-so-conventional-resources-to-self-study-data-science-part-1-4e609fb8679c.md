# 第 1 部分:学习数据科学编程的资源

# 计算机编程语言

如果你以前接触过任何计算机语言，我认为你不需要一本全面的 python 教科书来学习数据科学的 python。

在为数据科学学习 python 时，不要被深入钻研 python 编程和成为一名全面的 python 开发人员所淹没，重要的是要专注于学习语法和其他 python 基础知识，如列表理解、映射、归约、过滤等。让你开始。两个免费的在线资源可以帮你做到这一点，分别是[Python like your mean](https://www.pythonlikeyoumeanit.com/#)、 [Learn python](https://www.learnpython.org/) 和 [Mode](https://mode.com/python-tutorial/) 。

练习 Python 语法和基本操作，可以利用 Pynative [Python 练习题](https://pynative.com/python-exercises-with-solutions/)、[练习 Python](http://www.practicepython.org/) 、 [Hackerrank](https://www.hackerrank.com/domains/python/py-introduction/difficulty/all/page/1) 。

一旦你熟悉了 Python 基础知识，要学习 Python 中的数据操作、数据预处理和数据可视化，投资一本完整的书而不依赖在线资源是一个更好的主意。 [Python 数据科学手册](https://github.com/jakevdp/PythonDataScienceHandbook)(免费 PDF)第 2-4 章和[用于数据分析的 Python](https://www.amazon.com/Python-Data-Analysis-Wrangling-IPython/dp/1449319793)，由熊猫创始人本人执笔。如果你要从这两本书中选择一本，你的选择应该取决于你对熊猫和熊猫的了解程度。后者更深入高级 Numpy 和高级熊猫。

Udemy 有很好的[课程](https://www.udemy.com/course/learn-data-science-deep-learning-machine-learning-nlp-r/)，涵盖了 python 的基础知识和一些使用 python 的数据预处理。看看这个。警告:只是一切的基础。

# 结构化查询语言

到目前为止，您可能已经听到“数据科学世界”在向您宣传 SQL 的重要性。

在深入 SQL 之前，建议先具备 [*关系数据库系统*](https://www.tutorialspoint.com/sql/sql-rdbms-concepts.htm) 的基础知识。

[*SQL Server 初学者教程播放列表*](https://www.youtube.com/watch?v=7GVFYt6_ZFM&list=PL08903FB7ACA1C2FB) *说到 SQL 概念，是*的守护者。如果您完全不熟悉 SQL，或者如果您想刷新基础知识并学习一些高级概念，这是一个非常棒的教程(总共 150 个视频)。(可选零件 18–21、29–33)。

你可能会觉得，播放列表的一个缺点是，播放列表的组织不够完善。如果你想细分主题并以更有条理的方式观看，这篇文章可能会帮助你的计划。

[*SQL 面试问答*](https://www.youtube.com/watch?v=fvPddKyHxpQ&list=PL6n9fhu94yhXcztdLO7i6mdyaegC8CJwR) 是以上系列的延续，讲解了最常被问到的面试问题和概念。

大多数人在刚开始的时候很难掌握 Windows 函数的概念。这篇[文章](/@sundaskhalid/sql-window-function-vs-group-by-b246d14223d2)给出了这个概念的直观解释，并且解释了它与按功能分组的不同之处。 [Kaggle Advanced SQL](https://www.kaggle.com/learn/advanced-sql) 是一个简短的课程，很好地解释了这个概念以及其他一些高级概念，比如查询优化和索引。
