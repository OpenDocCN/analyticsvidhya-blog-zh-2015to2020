## 泡菜

*【pickle】*是将 Python 对象层次转换成*字节流*的过程，*【unpickle】*是逆操作，将字节流(来自二进制文件或类字节对象)转换回对象层次。(来源:https://docs.python.org/3/library/pickle.html[)](https://docs.python.org/))

它使得代码**与机器无关**，我们只需要解开文件并引用该文件就可以得到想要的输出。

## Rest API

API 是一个应用程序编程接口。它是一组允许程序相互对话的规则。开发人员在服务器上创建 API，并允许客户端与之对话。REST 决定了 API 的外观。它代表“代表性状态转移”它是开发人员在创建 API 时遵循的一组规则。

在接下来的教程中，我们将使用 ***Flask*** *创建一个 Rest API，这是一个用 Python 编写的微型 web 框架。*

## 云托管

虚拟主机是一种允许组织和个人在互联网上发布网站或网页的服务。

我在不同的云平台上托管过我的 Rest APIs，比如*谷歌云平台、微软 Azure 和 Heroku* ，我个人觉得 ***Heroku*** 使用起来简单方便，而且它还为每个账户提供了一个免费的网站托管。

我确信这是在互联网上部署您的模型的最方便的方式。你不需要额外的订阅，也不需要额外的编程语言来实现它，Python 为你做了一切。
