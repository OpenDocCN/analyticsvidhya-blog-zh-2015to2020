什么是 JWT？JWT 代表 JSON 网络令牌。

为什么是 JWT？JWT 定义了一种紧凑的、自包含的方式，以 JSON 对象的形式在各方之间安全地传输信息。可以使用它的一个例子是授权。

今天我将介绍如何使用 JWT 通过 Ruby on Rails 后端和 React 前端设置登录。
