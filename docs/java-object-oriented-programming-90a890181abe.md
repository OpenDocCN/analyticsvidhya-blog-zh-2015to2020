## 什么是超级关键词？

java 中的 super 关键字是一个引用变量，用于引用父类对象。因此，超级关键字有 3 个主要用途。

> 1.用于引用直接父类实例变量。
> 2。用于调用直接父类方法。
> 3。用于调用直接父类构造函数

## 1.用于引用直接父类实例变量。

如果父类和子类有相同的字段，我们可以使用 super 关键字来访问父类的数据或字段。
