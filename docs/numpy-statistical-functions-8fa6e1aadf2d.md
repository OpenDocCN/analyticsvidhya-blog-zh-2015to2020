# 统计功能

## 要查找平均值和方差:

mean:这将返回沿指定轴的算术平均值。

median:这将返回沿指定轴的中间值。

average:这将返回沿指定轴的加权平均值。

std:这将返回沿指定轴的标准偏差。

var:这将返回沿指定轴的方差。
