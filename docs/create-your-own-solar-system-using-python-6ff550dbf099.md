## 想象太阳系

我首先想到的是简单地使用 3D 散点图，为太阳和行星创建自定义大小的标记。然而，这并不理想，因为当放大或缩小时，标记保持相同的大小。

这导致了不必要的影响，如土星环在行星内部，一旦缩小，太阳吞没了所有的岩石内行星。这意味着我必须找到一个不同的解决方案来显示太阳和行星。

幸运的是，Plotly 还提供了一个 3D 表面图，我可以用它来绘制天体。然而，这需要画几个球，每个球的大小和颜色都不同。
