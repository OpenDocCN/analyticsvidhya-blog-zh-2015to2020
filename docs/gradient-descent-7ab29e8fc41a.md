# 什么是梯度下降？

*   GD 是一种在其他算法的成本函数上使用的算法，以便通过获得函数系数的梯度(即获得成本函数相对于其每个系数的导数)来最小化它。
*   通过运行 GD，我们可以获得实现最小成本(误差)值的系统系数的最佳值。
*   这些系数由机器学习模型在值的预测过程中使用。
