对于初学者来说，数据可视化似乎令人望而生畏，最好的学习方法是通过迭代和实践。但是，您可以遵循一些有用的规则和标准来帮助您在工作中创建有效的数据可视化。

> 首先，你必须创建一个一致的品牌。

一致的个人品牌对你的团队或办公室至关重要。如果你的团队的数据可视化有一个强大的品牌，你的利益相关者可以立即认出你的团队的工作。随着时间的推移，创建一个强大的品牌将有助于建立你的团队的信誉，并在更大的组织中建立你的工作。以下是你在创建团队品牌时应该考虑的一些要点:

**建立一致的格式** —您必须在您的团队制作的所有报告和仪表板之间建立一致的格式。首先，许多公司都有一本品牌手册——这些手册给出了具体配色方案、字体和标志的指导方针，供你使用。这本品牌手册对你的团队建立指导方针非常有帮助。

创建一个有凝聚力的标题——另一个需要标准化的格式是可视化的标题。在我的公司，我们将徽标放在左边，公司名称、可视化标题、日期和数据源放在右边。我们也经常在可视化的顶部放一个大标题和小描述。这种形式只是一个例子，许多设计都可以做得很好——最重要的是保持设计在团队中的凝聚力。

接下来，决定使用哪种工具来可视化数据。

如今工具层出不穷，很难知道使用哪种类型的软件进行可视化。幸运的是，有一些简单的标准可以帮助你做决定。

**使用 Excel 生成静态报告** — Excel 是一个强大的分析原始数据源的工具，但它也可以用于数据可视化。如果您需要创建一个永远不会改变的静态可视化，Excel 是一个很好的选择。Excel 也是创建视觉上吸引人的数据点列表的最佳选择。例如，如果我需要发送一份在市场营销活动中注册的客户名单，我会使用 Excel。

**在 Power BI 和 Tableau 中创建交互式仪表板** —如果您的最终用户需要能够操作或分析数据，仪表板是合适的工具。Power BI 和 Tableau 等软件允许用户创建能够讲述精彩故事的交互式仪表盘。如果你想要更好的可视化，你可以在技术上使用 Tableau 和 Power BI 来生成静态报告，但是真正的强大之处在于这些产品的交互性。如果你是数据可视化的新手，我会推荐 Power BI 而不是 Tableau——Power BI 是微软的产品，所以它的外观和感觉与你已经习惯的产品相似。它比 Tableau 使用起来更简单，学习曲线也更短。

**在 PowerPoint 中展示最终结果—** 一旦您完成了数据分析，如果您需要向利益相关者讲述故事，请使用 PowerPoint。幻灯片可以让你灵活地讲述一个连贯有趣的故事。

> 最后，选择您的数据可视化类型。

不幸的是，对于数据可视化来说，没有完美的公式，你的很多效率将来自于实践和学习什么是有效的，什么是失败的。但是，有一些最佳实践可以帮助您选择特定的数据可视化工具。

使用**折线图**分析一段时间内的趋势数据

使用**条形图**比较变量

使用**饼图/圆环图**显示整体的一部分。我建议每个馅饼不要超过四片

使用**地图**可视化物理位置。**热图**可以快速显示客户或竞争对手的高度集中。**气泡图**可以显示起点内有多少特定变量(例如，在潜在新扩张点的半径内有多少竞争对手？)

使用**表格**显示数据列表。当你的利益相关者需要看到清晰详细的数据点时，表格是很重要的。

最后，尽可能使数据可视化为**清晰**和**简单**。在一天结束的时候，没有人想学习如何使用仪表板或报告的教程。
