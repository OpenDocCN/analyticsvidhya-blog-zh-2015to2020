## 什么是 Seaborn？

Seaborn 是一个构建在 matplotlib 之上的不可思议的 Python 数据可视化库。它提供了一个高层次的界面来绘制有吸引力的和信息丰富的统计图形。

Seaborn 用于可视化随机分布。
