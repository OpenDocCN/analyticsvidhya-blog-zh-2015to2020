var(X，D)=[(6–12)+(10–12)+(20–12)]/(3–1)= 52

然后乘以权重:52 x 0.3 =15.6

2.遮蔽

平均值= (8 + 3+ 12)/3 =7.6

var(X，D)=[(8–7.6)+(3–7.6)+(12–7.6)]/(3–1)= 20.34

乘以权重后:20.34 x 0.3 = 6.102

3.下雨的

平均值= (2 + 14+ 19 + 8)/4 =10.75

var(X，D)=
[(2–10.75)+(14–10.75)+(19–10.75)+(8–10.75)]/(4–1)= 53.91

乘以权重后:53.91 x 0.4 = 21.564
